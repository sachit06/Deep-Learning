There are a total of three files present in the folder.
1. sachit_new.py - This python file has the implementation of model training which is done in google colab.
The further details regarding path locations are found in the comments of sachit.py file itself. Please run this file in colab itself.
2.weights.h5 -  This file has the weights of the final model which is trained on the given training data and can be used to evaluate on the test set.
3.test_script_Sachit : This file should be used to test the model. The details regarding the paths for model and test data are specified in the file itself.
You need to load weights.h5 model and test set in specified locations to test. It gives the final test accuracy as the output. 
You can run this file in colab or in your local system but make sure the path you give is correct.
